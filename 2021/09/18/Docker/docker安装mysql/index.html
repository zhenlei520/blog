docker安装redis

1. 下载镜像

docker pull mysql:latest

3. 查看镜像

docker images

4. 使用镜像创建容器

docker run --name mysql -v /E/ProgramFiles/Database/Docker-Mysql/conf:/etc/mysql/conf.d -v /E/ProgramFiles/Database/Docker-Mysql/logs:/logs -v /E/ProgramFiles/Database/Docker-Mysql/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=root -d -i -p 3306:3306 mysql

-p   3306:3306 :将容器的3306端口映射到主机的3306端口
-v   /E/ProgramFiles/Database/Docker-Mysql/data:/var/lib/mysql :将主机中当前目录下的data挂载到容器的/mysql
